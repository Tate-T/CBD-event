Протокол заходів і зустрічей

1. Планування

  • Менеджер контенту
  • Укладення договору
  • Інфолінія / дизайн
  • Бюджет
  • Попередня реєстрація
  • Запрошення - рекламна кампанія
  • Вибір дати/часу
  • Вибір теми і інфо лінія
  • Пошук таланту

2. Логістика

  • Політ / подорож
  • Оренда залу
  • Процеси харчування / бізнес-ланчі
  • Програмні продукти
  • Друк
  • Правила обслуговування
  • Програма дій
  • Фінанси
  • Звіти / інструкції / листування

3. Виконання

  • Підтримка учасників
  • Аудіо / відео обладнання
  • Обслуговування
  • Друк
  • Маркетинг
  • Фотозйомка
  • Реєстрація учасників
  • Модерація
  • Відповіді на запитання
  • Зворотний зв'язок

4. Менеджмент

  • Управління контрактами
  • Управління партнерами
  • Управління логістикою
  • Управління проектом
  • Ведення подій
  • Управління взаємодією
  • Нагляд / контроль
  • Звітність
  • Робота з фідбеком
